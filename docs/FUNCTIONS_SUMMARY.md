# ملخص الوظائف - نظام الصيانة التنبؤية للمركبات

## 1. MaintenanceContext - إدارة البيانات المركزية

### البيانات المخزنة
| الحقل | النوع | الوصف |
|------|------|-------|
| currentMileage | number | قراءة العداد الحالية (كم) |
| oilChangeInterval | number | المسافة لتغيير الزيت (كم) |
| lastOilChange | OilChange? | آخر تغيير زيت |
| oilChangeHistory | OilChange[] | سجل كامل للتغييرات |

### الوظائف (API)
| الوظيفة | المدخلات | الوظيفة |
|---------|----------|---------|
| updateCurrentMileage | mileage: number | تحديث قراءة العداد |
| updateOilChangeInterval | interval: 1000-50000 | تحديث مسافة تغيير الزيت |
| recordOilChange | notes?: string | تسجيل تغيير زيت جديد |
| getMileageSinceLastOilChange | - | المسافة منذ آخر تغيير |
| getRemainingMileage | - | المسافة المتبقية للتغيير |
| getOilLifePercentage | - | نسبة عمر الزيت 0-100% |
| getOilStatus | - | NORMAL / WARNING / CRITICAL |

---

## 2. Dashboard - لوحة التحكم

### الوظائف
- عرض 4 مستشعرات: زيت المحرك، حرارة، بطارية، زيت القير
- محاكاة تحديث كل 3 ثوان (±1 عشوائي)
- شريط الحالة العامة (حرج/تحذير/طبيعي)
- عرض العداد + زر تحديث + زر تسجيل تغيير زيت
- OilChangeCard (شريط تقدم + إحصائيات)
- شبكة 2×2 للمستشعرات
- ملخص: عدد طبيعي/تحذير/حرج
- MileageUpdateDialog و OilChangeDialog

---

## 3. Alerts - التنبيهات

### الوظائف
- قائمة تنبيهات (نشطة / مُطّلع عليها)
- مستويات: CRITICAL, HIGH, MEDIUM, LOW
- handleAcknowledge(alertId)
- Modal تفاصيل + إجراء مقترح
- formatTimestamp (وقت نسبي عربي)

---

## 4. History - السجل التاريخي

### الوظائف
- generateHistoricalData() - 24 نقطة محاكاة
- فلتر المستشعر (all/oil/temp/battery/trans)
- فلتر الفترة (today/week/month)
- إحصائيات: متوسط، أعلى، أقل
- رسم بياني خطي (Recharts → fl_chart)
- جدول آخر 8 صفوف

---

## 5. Reports - التقارير

### الوظائف
- بطاقات ملخص شهري (4)
- رسم أعمدة أسبوعي
- رسم دائري توزيع الحالات
- متوسطات شهرية (4 بطاقات)
- تنبيهات حسب النوع + اتجاه
- تصدير PDF (مستقبلي)

---

## 6. Settings - الإعدادات

### الوظائف
- تعديل قراءة العداد
- تعديل مسافة تغيير الزيت
- وحدة الحرارة (C/F)
- فترة التحديث (1,3,5,10 ثواني)
- الوضع الليلي
- الإشعارات + التنبيهات الحرجة فقط
- إعادة الاتصال التلقائي
- مدة حفظ البيانات (7-90 يوم)
- نسخ احتياطي / استعادة / مسح

---

## 7. MaintenanceLog - سجل الصيانة

### الوظائف
- 4 بطاقات ملخص
- Timeline تغيير الزيت
- تصدير السجل
- عرض تفاصيل كل سجل

---

## 8. المكونات المشتركة

| المكون | الوظيفة |
|--------|---------|
| RootLayout | Header + BottomNav + Outlet |
| SensorCard | بطاقة مستشعر (قيمة، حالة، اتجاه) |
| OilChangeCard | شريط تقدم + إحصائيات الزيت |
| OilChangeDialog | تسجيل تغيير زيت |
| MileageUpdateDialog | تحديث قراءة العداد |
